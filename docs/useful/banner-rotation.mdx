---
title: Ротация баннера
description: Обновление баннера при срабатывании событий на сервере
---

Функция **"Ротация баннера"** позволяет дополнить баннер вашего сервера дополнительной информацией. Поскольку эта функция использует тот же редактор изображений, что и [Приветственные сообщения](../general-settings.mdx#greeting-message), область его применения в баннерах очень широкая.

Изображение обновляется при срабатывании одного из следующих событий:

-   Пользователь присоединился к серверу
-   Пользователь покинул сервер
-   Сообщение отправлено
-   Голосовое подключение
-   Голосовое отключение

:::note Ограничения

Бесплатный план позволяет обновлять баннер только раз в час. С подпиской **Lacuna Diamond** этот интервал сокращается до 2 минут. Тем не менее, для обновления изображения должно сработать одно из вышеперечисленных событий.

:::

## Основные параметры \{#base-settings}

-   **Цвет фона изображения**: определяет цвет фона баннера, если фоновое изображение отсутствует.
-   **Ссылка на фоновое изображение**: устанавливает фоновое изображение. Поддерживаются как [заменители](../useful/replacers/data-types/README.mdx), возвращающие ссылку на изображение (например, `{ guild.banner }`), так и изображения с доменов `cdn.discordapp.com`, `media.discordapp.net` и `i.imgur.com`.

## Элементы изображения \{#image-elements}

Элементом (слоем) баннера может быть текст или изображение. Для обоих типов существуют общие параметры:

-   позиция по оси X;
-   позиция по оси Y;
-   высота;
-   ширина.

### Изображение \{#image-elements-image}

Для слоя типа "Изображение" предусмотрены следующие параметры:

-   ссылка на изображение ([заменитель](../useful/replacers/data-types/README.mdx), возвращающий ссылку на изображение либо изображение с доменов `cdn.discordapp.com` и `i.imgur.com`);
-   радиус краёв.

### Текст \{#image-elements-text}

Для слоя типа "Текст" предусмотрены следующие параметры:

-   цвет текста;
-   размер (заголовок/подзаголовок/стандартный/нижний колонтитул);
-   стиль (стандартный/курсив);
-   преобразование (верхний/нижний регистр, с заглавной буквы);
-   оформление (подчеркнутый/зачеркнутый);
-   выравнивание (по центру или левому/правому краю).
