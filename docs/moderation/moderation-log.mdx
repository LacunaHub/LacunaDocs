## Что записывается в журнал модерации \{#what-is-written}

- Баны и разбаны
- Исключения с сервера
- Массовая очистка сообщений
- Выдача и снятие заглушений
- Выдача и снятие предупреждений

## Включение журнала модерации \{#activate-cases-log}

Нужно перейти в настройки сервера на сайте во вкладку **Модерирование** и выбрать канал в поле **Канал для случаев.**

![Выбор канала](static/img/moderation-log-select-channel.png)

Журнал модерации не будет работать до тех пор, пока вы не установите канал.

## Изменение причины случая \{#edit-case-reason}

Вы можете в любой момент изменить причину для случая в журнале. Для этого откройте канал, который назначен для отправки событий журнала модерации, и под нужным случаем нажмите кнопку "Изменить причину".

![Кнопка изменения причины](static/img/moderation-log-change-reason.png)

Введите нужную причину для случая модерации и нажмите "Отправить".

![Заполнение причины](static/img/moderation-log-fill-reason.png)

После чего вы увидите обновлённое сообщение с указанием причины.

![Случай модерации с указанной причиной](static/img/moderation-log-with-reason.png)

:::info Помните!

Вы можете изменить только последние 50 случаев в журнале.

:::
